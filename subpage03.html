PROJECT 01

대학생활 통합 서비스 기획 '캠퍼스 라이프 앱'

PROJECT OVERVIEW (핵심 기능)

대학생들이 일상에서 가장 자주 확인하는 시간표, 학점 계산, 오늘의 식단, 친구 목록 기능을 파편화 없이 하나의 모바일 화면 흐름 안에 직관적으로 배치하고 매끄럽게 정리한 올인원 통합 서비스입니다.

PROJECT INFOS

공모전 기획 모바일 서비스 UI/UX 대학생 타겟 유저 카드 UX 디자인
STRATEGY 01
아이디어의 시작

대학생들이 하루에도 수차례 교내 웹사이트와 커뮤니티를 오가며 개별적으로 확인해야 했던 번거로운 시간표, 학점, 식단, 친구 정보들을 한 화면 안에서 딜레이 없이 신속하게 연동하고 파악할 수 있도록 통합형 앱 서비스를 구상하여 기획했습니다.

STRATEGY 02
화면 구성 및 인터페이스

효율적인 시각적 흐름을 구축하기 위해 좌측 간이 메뉴와 하단 고정 내비게이션 바를 유기적으로 동시 배치했습니다. 사용자가 주요 핵심 기능 간에 막힘없이 전환 이동하기 쉽도록 설계했으며, 복잡할 수 있는 시간표 영역은 과목별 고유의 파스텔 색상 코드를 부여하여 시인성을 극대화했습니다.

STRATEGY 03
사용자 경험(UX) 고도화

학점 계산기, 오늘의 학식 식단 현황, 실시간 친구 검색 등 흩어지기 쉬운 부가 위젯 기능들을 규격화된 입체감 있는 카드(Card Layout) UI 형태로 정형화했습니다. 이를 통해 사용자가 앱을 실행하자마자 본인에게 필요한 개인화 정보를 한눈에 스캔하여 파악할 수 있도록 인지 부하를 줄였습니다.
